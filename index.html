vue框架

2019年10月11日
前言 vue 官方

Vue 是基于 MVVM 设计模式的渐进式的前端 js 框架
渐进式: 可以有选择的逐步使用框架中的组件,可以配合其他组件 js 库使用; 全家桶: 必须全盘使用所有组件
前端 js 框架，不需要使用 nodejs 但是需要使用到中的 babel 编译 ex6 中的浏览器不认识的语音，
侧重于以数据操作为主的前端项目开发

vue框架 对比 jquery 原生 dom

  • 原生 dom 优点 不用导入任何的框架 插件，跨平台好，在操作 dom 时相对麻烦，

  • jquery 中，相对于 原生 dom 更加方便，对 dom 的每一个 api 进行了简化，但是任然需操作到 dom

  • 框架 优点 彻底简化了开发的步骤，基本上不需要操作 dom， 这种优化，需要转变之前的观念...
